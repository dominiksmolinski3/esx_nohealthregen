<h1 align='center'>[ESX] nohealthregen </a></h1><p align='center'>

simple resource created to disable health regeneration in GTA V FiveM, ESX framework

## Dependencies
ESX Legacy

## Installation
Download from 'Releases' or manually.

Rename from 'esx_nohealthregen-main' (or smth other) to 'esx_nohealthregen' 

Put the 'esx_nohealthregen' into 'resources' in your server.

Add 'ensure esx_nohealthregen' or 'start esx_nohealthregen' whichever you please to the server.cfg

All done! Contact me if need help.
